## 1. Hypotheses (pre-registered)

We pre-register confirmatory tests of the following hypotheses, derived from the pilot study (Acharjee & Nowshin, MGRS 2026 poster).

**Primary:**

- **H1.** Across leadership-coded prompts (CEO, founder, manager, leader, executive), DALL·E 3 produces ≥ 80% male-coded outputs.
- **H2.** Across care-coded prompts (nurse, teacher, caregiver, receptionist, social worker), DALL·E 3 produces ≥ 80% female-coded outputs.
- **H3.** In a forced-choice recognition task with 4 alternatives (3 demographically diverse, 1 DALL·E default), participants select the DALL·E default at a rate significantly above chance (25%) for the CEO prompt (one-tailed binomial; significance threshold α = .05 / 9 Bonferroni).
- **H4.** The same effect holds for at least one care-coded prompt with the same correction.

**Secondary (exploratory/confirmatory):**

- **H5.** Participant gender modulates H3/H4 selection rate. Specifically: male participants will select DALL·E defaults at a higher rate than female participants across all 9 prompts (Mann-Whitney U, two-tailed).
- **H6.** Participants asked "which would you actually use" (intent-to-redistribute) select DALL·E defaults at a rate significantly above the recognition rate for at least the CEO prompt (paired-difference test).

**Cross-model:**

- **H7.** Stage-1 stereotype rates (H1, H2) replicate across at least one open-source model (Stable Diffusion XL) and one closed-source competitor (Imagen 3 or Midjourney v7).

## 2. Sampling plan

**Stage 1 (image generation).** 20 images per prompt × 10 prompts × 3 models = **600 images total**. Coded by 3 independent annotators recruited via Prolific (vetted CV/HCI background, paid ≥ $15/hour). Inter-annotator agreement reported; disagreements adjudicated by a 4th senior coder.

**Stage 2 (recognition survey).** Target n = **120 participants** recruited via Prolific. Stratified for balanced gender (male/female/non-binary in pre-determined ratios) and balanced academic/professional background (STEM / arts / business / humanities) to address the gender-discipline confound observed in the pilot.

**Stage 3 (intent-to-redistribute).** Same 120 participants. Within-subject design: each participant answers both "which best represents this role" (recognition) and "which would you use in a presentation" (intent) for each role.

**Stopping rule.** Data collection halts at n = 120 confirmed completions OR after 4 weeks, whichever first. No early stopping for "interim looks."

## 3. Power analysis

For H3 (CEO recognition rate above chance), the pilot effect was 71% vs. 25% chance, OR = 7.20. With n = 120 and α = .05/9 = .0056 (Bonferroni), one-tailed binomial against chance, power ≥ .99 for any true effect ≥ 50%.

For H5 (gender moderation), Cohen's *d* = 1.48 in the pilot. With n ≥ 40 per group and α = .05, two-tailed Mann-Whitney U, power ≥ .95 for any true *d* ≥ 0.5.

The replication is over-powered relative to the pilot; this is intentional, since pilot effects often regress.

## 4. Variables

**Independent:**
- Prompt category (leadership / care / STEM / neutral-human; pre-defined per `PROMPT_GUIDE.md` v1).
- Model (DALL·E 3 / Stable Diffusion XL / Imagen 3 or Midjourney v7).
- Participant gender (self-reported, free-form bucketed to male/female/non-binary/decline).
- Participant academic-or-professional discipline (self-reported).
- Question framing (recognition / intent-to-redistribute).

**Dependent:**
- Annotator-coded gender, race, attire, setting per image (3 independent codings; majority + agreement statistic).
- Participant choice (4-option forced choice, alternatives randomized in display order).
- Participant time-on-task per item (instrument-side telemetry).

**Control:**
- Display order of alternatives (counterbalanced via Latin square).
- Image polish (style-matched alternatives generated by the same model with diversity-injecting prompt expansion — addresses the AI-quality confound).
- Participant attention checks (3 distributed throughout the survey; failures excluded pre-analysis).

## 5. Analysis plan

All analyses will be run by both authors independently; numerical disagreement resolved by re-running a single canonical script (committed to OSF before data collection begins).

**For H1, H2 (image-side):** Per-prompt majority-coded gender. Reported as percentages with 95% Wilson confidence intervals. Report inter-annotator Krippendorff's α.

**For H3, H4 (recognition):** One-tailed binomial against chance (25%). Bonferroni correction across 9 prompts (α = .0056). Report effect size (Cohen's *h*), 95% CI on selection rate.

**For H5 (gender moderation):** Mann-Whitney U test on per-participant DALL·E-default-count (max 9). Report Cohen's *d*, 95% CI, and the per-prompt breakdown.

**For H6 (intent vs. recognition):** McNemar's test (paired binary outcome) on per-participant per-prompt comparison. Bonferroni across 9 prompts.

**For H7 (cross-model):** Same H1/H2 procedure per model. Report between-model differences with chi-square contingency tests.

## 6. Exclusion criteria

Participants excluded if:
- Failed ≥ 2 of 3 attention checks.
- Completed survey in less than 2 minutes (insufficient deliberation).
- Self-reported "I clicked randomly" on debrief item.

These criteria were finalized before data collection.
